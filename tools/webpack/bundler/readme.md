# 简易打包工具编写

> npm install @babel/parser --save-dev  //babel解析器,可以将代码转换为AST对象

> npm install @babel/traverse --save  //遍历AST节点，有助于我们查找

> npm install @babel/core --save  //babel 核心模块

> npm install @babel/preset-env --save

[source](./bundle.js)
